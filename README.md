# smartexlab-task
smartexlab test task for Javascript

#Requirements
- ECMAScript 6
- angular 1.5.x
- node 4.6.x
- npm 4.0.x
- bower 1.8.x
- gulp 3.9.x
- jsHint

#How to start

1. Clone repository
1. open smartexlab-task folder: `$ cd smartexlab-task`
1. execute `$ npm install`
1. open `index.html`
